Task 2: Churn Count Visualization

This task focuses on visualizing customer churn by comparing the number of churned and active customers using Microsoft Excel. A column chart was created to provide a clear representation of customer retention and churn, enabling quick interpretation of customer status distribution.

Key Activities
Calculated the total number of churned and active customers.
Created a column chart to compare customer churn status.
Added data labels, chart title, and axis labels for better readability.
Analyzed customer retention and churn distribution.
Derived business insights to support customer retention strategies.
Skills Demonstrated
Microsoft Excel
Data Visualization
Column Chart Creation
Data Analysis
Customer Churn Analysis
Business Insight Generation
Chart Formatting
Key Findings
1,869 customers have churned, while 5,174 customers remain active.
The customer retention rate is 73.46%, compared to a 26.54% churn rate.
The visualization highlights a strong active customer base while emphasizing opportunities to improve customer retention.
Tools Used
Microsoft Excel / WPS Sheets

This visualization provides a foundational understanding of customer churn patterns and supports data-driven decision-making for customer retention initiatives.
